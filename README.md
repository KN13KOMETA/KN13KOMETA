  <summary><h2>About me</h2></summary>

  <p>
    I started learning programming in 2018-19. That's when I was mostly writing Discord bots and created <a href="https://github.com/KN13KOMETA/ASOUL">ASOUL</a> (which I later rewrote in TypeScript and named <a href="https://github.com/ClintFlames/asoul-r">asoul-r</a>). After that, I tried making some games in Unity/Godot. In 2022, I found out about Haxe from Dead Cells. I liked the language very much (it's still the most pleasurable language to write in), but sadly most of its ecosystem lacks documentation and is deprecated. So I decided to switch. I tried C++ but didn't like a lot of legacy things it carries. Then I tried Rust and wrote <a href="https://github.com/KN13KOMETA/simple-rust-snake">simple-rust-snake</a>, but didn't like how it's tied too much to its crates and got tired of fighting with the borrow checker. So I switched over to C and am currently writing a <a href="https://github.com/KN13KOMETA/c_text_rpg">c_text_rpg</a>. The only thing I dislike about C is how much time I wasted setting up cross-compiling and the build system. After releasing v3, I am planning to learn some Zig to use it as a build system, as I already use it as a cross-compiler.

  </p>
